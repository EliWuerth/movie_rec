# 🎬 Movie Recommendation Site

A responsive web application that helps users discover, browse, and save their favorite movies. Built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- 🏠 **Home Page** - Welcome screen with featured content
- 🎥 **Movie Browse** - Explore and search through a movie catalog
- ❤️ **Favorites** - Save and manage your favorite movies
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices
- 🎨 **Modern UI** - Clean and intuitive user interface
- 💾 **Persistent Storage** - Your favorites are saved locally

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server setup required - runs locally

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/EliWuerth/movie_rec.git
   cd movie_rec
   ```

2. **Open in browser**
   - Simply open `index.html` in your favorite web browser
   - Or use a local server:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Python 2
     python -m SimpleHTTPServer 8000
     
     # Node.js (with http-server)
     npx http-server
     ```
   - Visit `http://localhost:8000`

## 💻 Usage

1. **Browse Movies** - Navigate to the movies page to explore the catalog
2. **Add to Favorites** - Click the heart icon to save movies you like
3. **View Favorites** - Visit the favorites page to see your saved movies
4. **Learn More** - Check the about page for project details
